# 🎓 ExamCoach — Personalized AI Entrance Exam Coach

## 📁 File Structure

```
ExamCoach/
├── main.py                    ← Entry point — run this!
├── config.py                  ← API keys (OpenAI + Gemini)
├── database.py                ← SQLite schema
├── index.html                 ← Complete frontend (served at /)
├── requirements.txt
├── START.bat                  ← Windows: double-click to run
├── start.sh                   ← Mac/Linux: bash start.sh
│
├── routes/
│   └── api.py                 ← All REST API endpoints
│
├── services/
│   ├── llm_service.py         ← OpenAI + Gemini integration
│   └── analytics.py           ← Weak topic detection, dashboard data
│
└── questions/
    └── question_bank.py       ← 50+ curated questions for all exams
```

## 🚀 How to Run

### Windows
```
Double-click START.bat
```

### Mac / Linux
```bash
bash start.sh
```

### Manual
```bash
pip install -r requirements.txt
python main.py
# Then open: http://localhost:8000
```

## 🔑 API Keys

Already configured in `config.py`:
- **OpenAI GPT-4o** — Error analysis, AI hints, revision plans
- **Google Gemini** — Study material recommendations

## ✨ Features

- 🧠 **AI Quiz** with timed questions, hints, and instant explanations
- 🎉 **Confetti celebration** when you score 75%+
- 🤖 **AI error analysis** after every quiz
- 📅 **7-day personalized revision plan** generated by GPT-4o
- 📚 **Study materials** curated by Gemini for weak topics
- 📊 **Dashboard** with charts, weak topic heatmap, score trend
- 🗓️ **Daily progress log**

## 🌐 API Endpoints

| Method | Endpoint | Description |
|--------|---------|-------------|
| POST | `/api/students` | Register student |
| GET | `/api/questions/{exam}` | Get quiz questions |
| POST | `/api/quiz/submit` | Submit quiz + AI analysis |
| GET | `/api/dashboard/{id}` | Full dashboard data |
| POST | `/api/plan/{id}/generate` | Generate 7-day plan |
| GET | `/api/materials` | Get Gemini study materials |
| GET | `/api/quiz/history/{id}` | Quiz history |
